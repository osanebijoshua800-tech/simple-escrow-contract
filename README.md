# Simple Escrow Smart Contract

## Overview
A Solidity-based escrow contract that securely holds ETH between a payer and a payee until payment is released or refunded.
Secure Web3 Escrow Payments

Milestone-based token escrow with dispute resolution and wallet integration.
## Features
- Secure ETH holding
- Controlled payment release
- Refund mechanism
- Event logging
- Reentrancy-safe transfer

## Use Cases
- Freelance payments
- Service agreements
- Web3 transactions
  
# how it works
1. Connect wallet
2. Approve tokens
3. Deposit funds
4. Release milestones or resolve disputes
